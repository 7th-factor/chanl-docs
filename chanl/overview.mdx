# Chanl Platform Overview

Chanl is a comprehensive platform for testing, observing, and optimizing AI agents through structured scenarios and real-time monitoring.

## Platform Architecture

The platform follows a hierarchical structure where high-level features are built using foundational components:

- **Scenarios** are built using **Personas**, **Agents**, and **Scorecards**
- **Simulations** are the result of running scenarios
- **Analytics** provide insights across all components

## Getting Started

1. **Create Personas** - Define customer behavior patterns
2. **Configure Agents** - Set up your AI agents with prompts and tools
3. **Design Scenarios** - Create test situations using personas and agents
4. **Set Up Scorecards** - Define quality evaluation criteria
5. **Run Simulations** - Execute tests and gather results
6. **Analyze Results** - Use analytics to identify improvements
7. **Optimize** - Update prompts, tools, and configurations based on insights
