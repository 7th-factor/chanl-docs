---
title: 'Scenario Statistics'
openapi: 'GET /api/v1/workspaces/{workspace_id}/scenarios/stats'
---

View usage and performance statistics for all scenarios.

<RequestExample>

```bash cURL
curl -X GET "https://api.chanl.ai/api/v1/workspaces/ws_123/scenarios/stats" \
  -H "Authorization: Bearer <access_token>"
```

</RequestExample>

<ResponseExample>

```json Success Response
{
  "success": true,
  "data": {
    "stats": {
      "total_scenarios": 25,
      "active_scenarios": 18,
      "total_simulations": 1450,
      "avg_score_last_30_days": 84.2,
      "most_used_scenario": {
        "id": "scenario_123",
        "name": "Customer Service Inquiry",
        "simulation_count": 156
      }
    }
  }
}
```

</ResponseExample>
