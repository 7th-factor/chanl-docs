---
title: 'List Scenarios'
openapi: 'GET /api/v1/workspaces/{workspace_id}/scenarios'
---

Get all scenarios in a workspace with filtering and search capabilities.

Scenarios define test situations combining personas and agents for comprehensive evaluation.

<RequestExample>

```bash cURL
curl -X GET "https://api.chanl.ai/api/v1/workspaces/ws_123/scenarios" \
  -H "Authorization: Bearer <access_token>"
```

```javascript JavaScript
const response = await fetch(`https://api.chanl.ai/api/v1/workspaces/${workspaceId}/scenarios`, {
  headers: {
    'Authorization': 'Bearer ' + accessToken
  }
});

const { data } = await response.json();
console.log('Scenarios:', data.scenarios);
```

```python Python
import requests

response = requests.get(
    f'https://api.chanl.ai/api/v1/workspaces/{workspace_id}/scenarios',
    headers={'Authorization': f'Bearer {access_token}'}
)

scenarios = response.json()['data']['scenarios']
```

</RequestExample>
