---
title: 'Get Persona'
openapi: 'GET /api/v1/workspaces/{workspace_id}/personas/{id}'
---

Retrieve detailed information about a specific persona.

Includes full prompt text, usage statistics, and recent simulation performance data.

<RequestExample>

```bash cURL
curl -X GET "https://api.chanl.ai/api/v1/workspaces/ws_123/personas/persona_456" \
  -H "Authorization: Bearer <access_token>"
```

```javascript JavaScript
const response = await fetch(`https://api.chanl.ai/api/v1/workspaces/${workspaceId}/personas/${personaId}`, {
  headers: {
    'Authorization': 'Bearer ' + accessToken
  }
});

const { data } = await response.json();
console.log('Persona details:', data.persona);
```

```python Python
import requests

response = requests.get(
    f'https://api.chanl.ai/api/v1/workspaces/{workspace_id}/personas/{persona_id}',
    headers={'Authorization': f'Bearer {access_token}'}
)

persona = response.json()['data']['persona']
```

</RequestExample>
