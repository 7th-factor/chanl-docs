---
title: 'List Personas'
openapi: 'GET /api/v1/workspaces/{workspace_id}/personas'
---

Get all personas in a workspace with filtering and search capabilities.

Personas represent different customer behaviors and characteristics used in testing scenarios.

<RequestExample>

```bash cURL
curl -X GET "https://api.chanl.ai/api/v1/workspaces/ws_123/personas" \
  -H "Authorization: Bearer <access_token>"
```

```javascript JavaScript
const response = await fetch(`https://api.chanl.ai/api/v1/workspaces/${workspaceId}/personas`, {
  headers: {
    'Authorization': 'Bearer ' + accessToken
  }
});

const { data } = await response.json();
console.log('Personas:', data.personas);
```

```python Python
import requests

response = requests.get(
    f'https://api.chanl.ai/api/v1/workspaces/{workspace_id}/personas',
    headers={'Authorization': f'Bearer {access_token}'}
)

personas = response.json()['data']['personas']
```

## Query Parameters

<ParamField query="name" type="string">
  Search personas by name (partial match)
</ParamField>

<ParamField query="language" type="string">
  Filter by language code (e.g., "en", "es", "fr")
</ParamField>

<ParamField query="emotion" type="string">
  Filter by emotional state (frustrated, happy, confused, neutral, etc.)
</ParamField>

<ParamField query="difficulty" type="string">
  Filter by difficulty level (easy, medium, hard)
</ParamField>

<ParamField query="tags" type="string">
  Filter by tags (comma-separated)
</ParamField>

<ParamField query="createdBy" type="string">
  Filter by creator user ID
</ParamField>

<ParamField query="sortBy" type="string">
  Sort field (name, createdAt, updatedAt, usage)
</ParamField>

<ParamField query="sortOrder" type="string">
  Sort direction (asc, desc)
</ParamField>

<ParamField query="limit" type="number">
  Maximum number of results (default: 50, max: 100)
</ParamField>

<ParamField query="offset" type="number">
  Number of results to skip for pagination
</ParamField>

## Filtering Examples

<CodeGroup>

```bash Filter by Emotion
curl -X GET "https://api.chanl.ai/api/v1/workspaces/ws_123/personas?emotion=frustrated" \
  -H "Authorization: Bearer <access_token>"
```

```bash Filter by Tags
curl -X GET "https://api.chanl.ai/api/v1/workspaces/ws_123/personas?tags=support,billing" \
  -H "Authorization: Bearer <access_token>"
```

```bash Search by Name
curl -X GET "https://api.chanl.ai/api/v1/workspaces/ws_123/personas?name=angry" \
  -H "Authorization: Bearer <access_token>"
```

```bash Sort by Usage
curl -X GET "https://api.chanl.ai/api/v1/workspaces/ws_123/personas?sortBy=usage&sortOrder=desc" \
  -H "Authorization: Bearer <access_token>"
```

</CodeGroup>
