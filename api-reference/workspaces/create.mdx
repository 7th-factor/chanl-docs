---
title: 'Create Workspace'
openapi: 'POST /api/v1/workspaces'
---

Create a new workspace for organizing analytics and team collaboration.

The user creating the workspace will automatically become the owner with full administrative privileges.

<RequestExample>

```bash cURL
curl -X POST "https://api.chanl.ai/api/v1/workspaces" \
  -H "Authorization: Bearer <access_token>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Support Team Analytics",
    "slug": "support-team",
    "description": "Analytics workspace for customer support team"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.chanl.ai/api/v1/workspaces', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer ' + accessToken,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    name: 'Support Team Analytics',
    slug: 'support-team',
    description: 'Analytics workspace for customer support team'
  })
});

const { data } = await response.json();
console.log('New workspace:', data.workspace);
```

```python Python
import requests

response = requests.post(
    'https://api.chanl.ai/api/v1/workspaces',
    headers={'Authorization': f'Bearer {access_token}'},
    json={
        'name': 'Support Team Analytics',
        'slug': 'support-team',
        'description': 'Analytics workspace for customer support team'
    }
)

workspace = response.json()['data']['workspace']
```
