---
title: 'Get Workspace'
openapi: 'GET /api/v1/workspaces/{workspace_id}'
---

Retrieve detailed information about a specific workspace.

Returns comprehensive workspace details including settings, member count, and usage statistics.

<RequestExample>

```bash cURL
curl -X GET "https://api.chanl.ai/api/v1/workspaces/ws_123" \
  -H "Authorization: Bearer <access_token>"
```

```javascript JavaScript
const response = await fetch(`https://api.chanl.ai/api/v1/workspaces/${workspaceId}`, {
  headers: {
    'Authorization': 'Bearer ' + accessToken
  }
});

const { data } = await response.json();
console.log('Workspace details:', data.workspace);
```

```python Python
import requests

response = requests.get(
    f'https://api.chanl.ai/api/v1/workspaces/{workspace_id}',
    headers={'Authorization': f'Bearer {access_token}'}
)

workspace = response.json()['data']['workspace']
```

</RequestExample>
