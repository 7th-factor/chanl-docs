---
title: 'Update Workspace'
openapi: 'PUT /api/v1/workspaces/{workspace_id}'
---

Update workspace information and settings.

Only workspace owners and administrators can update workspace details. Members can only update specific settings they have permission to modify.

<RequestExample>

```bash cURL
curl -X PUT "https://api.chanl.ai/api/v1/workspaces/ws_123" \
  -H "Authorization: Bearer <access_token>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Acme Customer Analytics",
    "description": "Updated analytics workspace for customer support and sales teams"
  }'
```

```javascript JavaScript
const response = await fetch(`https://api.chanl.ai/api/v1/workspaces/${workspaceId}`, {
  method: 'PUT',
  headers: {
    'Authorization': 'Bearer ' + accessToken,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    name: 'Acme Customer Analytics',
    description: 'Updated analytics workspace for customer support and sales teams'
  })
});

const { data } = await response.json();
console.log('Updated workspace:', data.workspace);
```

```python Python
import requests

response = requests.put(
    f'https://api.chanl.ai/api/v1/workspaces/{workspace_id}',
    headers={'Authorization': f'Bearer {access_token}'},
    json={
        'name': 'Acme Customer Analytics',
        'description': 'Updated analytics workspace for customer support and sales teams'
    }
)

workspace = response.json()['data']['workspace']
```
