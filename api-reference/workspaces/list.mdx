---
title: 'List Workspaces'
openapi: 'GET /api/v1/workspaces'
---

Get all workspaces the authenticated user has access to.

Returns a list of workspaces with their basic information, user role, and permission levels.

<RequestExample>

```bash cURL
curl -X GET "https://api.chanl.ai/api/v1/workspaces" \
  -H "Authorization: Bearer <access_token>"
```

```javascript JavaScript
const response = await fetch('https://api.chanl.ai/api/v1/workspaces', {
  headers: {
    'Authorization': 'Bearer ' + accessToken
  }
});

const { data } = await response.json();
console.log('Workspaces:', data.workspaces);
```

```python Python
import requests

response = requests.get(
    'https://api.chanl.ai/api/v1/workspaces',
    headers={'Authorization': f'Bearer {access_token}'}
)

workspaces = response.json()['data']['workspaces']
```

</RequestExample>
