---
title: 'Delete Workspace'
openapi: 'DELETE /api/v1/workspaces/{workspace_id}'
---

Permanently delete a workspace and all its associated data.

<Warning>
This action is irreversible. All personas, scenarios, simulations, and analytics data will be permanently deleted.
</Warning>

Only workspace owners can delete workspaces. All members will lose access immediately.

<RequestExample>

```bash cURL
curl -X DELETE "https://api.chanl.ai/api/v1/workspaces/ws_123" \
  -H "Authorization: Bearer <access_token>" \
  -H "Content-Type: application/json" \
  -d '{
    "confirm": true
  }'
```

```javascript JavaScript
const response = await fetch(`https://api.chanl.ai/api/v1/workspaces/${workspaceId}`, {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer ' + accessToken,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    confirm: true
  })
});

const { data } = await response.json();
console.log('Workspace deleted:', data.message);
```

```python Python
import requests

response = requests.delete(
    f'https://api.chanl.ai/api/v1/workspaces/{workspace_id}',
    headers={'Authorization': f'Bearer {access_token}'},
    json={'confirm': True}
)

message = response.json()['data']['message']
```

</RequestExample>
