Manage workspace configuration and preferences.

Settings control workspace behavior, integrations, data retention, and notification preferences.

## Get Settings

<RequestExample>

```bash cURL
curl -X GET "https://api.chanl.ai/api/v1/workspaces/ws_123/settings" \
  -H "Authorization: Bearer <access_token>"
```

```javascript JavaScript
const response = await fetch(`https://api.chanl.ai/api/v1/workspaces/${workspaceId}/settings`, {
  headers: {
    'Authorization': 'Bearer ' + accessToken
  }
});

const { data } = await response.json();
console.log('Settings:', data.settings);
```

```python Python
import requests

response = requests.get(
    f'https://api.chanl.ai/api/v1/workspaces/{workspace_id}/settings',
    headers={'Authorization': f'Bearer {access_token}'}
)

settings = response.json()['data']['settings']
```

## Update Settings

<RequestExample>

```bash cURL
curl -X PUT "https://api.chanl.ai/api/v1/workspaces/ws_123/settings" \
  -H "Authorization: Bearer <access_token>" \
  -H "Content-Type: application/json" \
  -d '{
    "general": {
      "timezone": "America/Los_Angeles"
    },
    "data": {
      "retention_days": 180,
      "auto_analyze": true
    },
    "notifications": {
      "webhook_url": "https://newapp.com/webhooks/chanl",
      "daily_summary": false
    }
  }'
```

```javascript JavaScript
const response = await fetch(`https://api.chanl.ai/api/v1/workspaces/${workspaceId}/settings`, {
  method: 'PUT',
  headers: {
    'Authorization': 'Bearer ' + accessToken,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    general: {
      timezone: 'America/Los_Angeles'
    },
    data: {
      retention_days: 180,
      auto_analyze: true
    },
    notifications: {
      webhook_url: 'https://newapp.com/webhooks/chanl',
      daily_summary: false
    }
  })
});

const { data } = await response.json();
console.log('Settings updated:', data.settings);
```

```python Python
import requests

response = requests.put(
    f'https://api.chanl.ai/api/v1/workspaces/{workspace_id}/settings',
    headers={'Authorization': f'Bearer {access_token}'},

            'timezone': 'America/Los_Angeles'
        },
        'data': {
            'retention_days': 180,
            'auto_analyze': True

            'webhook_url': 'https://newapp.com/webhooks/chanl',

)

settings = response.json()['data']['settings']
```

## Setting Categories

### General Settings
- **timezone**: Workspace timezone (IANA timezone format)
- **date_format**: Date display format
- **time_format**: 12h or 24h time format
- **language**: Interface language code

### Data Settings
- **retention_days**: How long to keep simulation data (30-365 days)
- **auto_analyze**: Automatically analyze new simulations
- **backup_enabled**: Enable automatic backups
- **backup_frequency**: Backup frequency (daily, weekly, monthly)

### Notification Settings
- **email_enabled**: Enable email notifications
- **webhook_url**: URL for webhook notifications
- **webhook_events**: Array of events to send webhooks for
- **daily_summary**: Send daily analytics summary

### Integration Settings
- **vapi_enabled**: Enable VAPI integration
- **twilio_enabled**: Enable Twilio integration
- **slack_webhook**: Slack webhook URL for notifications

### Security Settings
- **ip_whitelist**: Array of allowed IP addresses/ranges
- **require_2fa**: Require two-factor authentication
- **session_timeout**: Session timeout in hours

## Webhook Events

Available webhook events:
- `simulation.completed` - Simulation finished successfully
- `simulation.failed` - Simulation encountered an error
- `simulation.started` - Simulation began execution
- `alert.triggered` - Alert condition was met
- `schedule.failed` - Scheduled execution failed
- `workspace.member_added` - New member joined workspace
- `workspace.member_removed` - Member left or was removed
