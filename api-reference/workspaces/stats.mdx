---
title: 'Workspace Statistics'
openapi: 'GET /api/v1/workspaces/{workspace_id}/stats'
---

Get comprehensive usage and performance statistics for a workspace.

Provides insights into workspace activity, resource usage, performance metrics, and trends over time.

<RequestExample>

```bash cURL
curl -X GET "https://api.chanl.ai/api/v1/workspaces/ws_123/stats" \
  -H "Authorization: Bearer <access_token>"
```

```javascript JavaScript
const response = await fetch(`https://api.chanl.ai/api/v1/workspaces/${workspaceId}/stats`, {
  headers: {
    'Authorization': 'Bearer ' + accessToken
  }
});

const { data } = await response.json();
console.log('Workspace stats:', data.stats);
```

```python Python
import requests

response = requests.get(
    f'https://api.chanl.ai/api/v1/workspaces/{workspace_id}/stats',
    headers={'Authorization': f'Bearer {access_token}'}
)

stats = response.json()['data']['stats']
```

## Statistics Categories

### Overview
High-level counts of workspace resources and activity.

### Performance
Quality metrics and scoring trends showing workspace performance over time.

### Activity
Usage patterns including simulation frequency, peak times, and activity trends.

### Top Performers
Best-performing scenarios, personas, and agents based on scores and usage.

### Resource Usage
Storage consumption, API usage, and plan limits.

### Trends
Historical data showing performance and activity changes over time.

### Alerts
Alert activity and the most frequently triggered alert conditions.

## Query Parameters

<ParamField query="period" type="string">
  Time period for statistics (7d, 30d, 90d) - default: 30d
</ParamField>

<ParamField query="include_trends" type="boolean">
  Include historical trend data - default: true
</ParamField>

<ParamField query="timezone" type="string">
  Timezone for date-based statistics - default: workspace timezone
</ParamField>

## Use Cases

- **Dashboard Analytics**: Power workspace overview dashboards
- **Performance Monitoring**: Track quality trends and identify issues
- **Resource Planning**: Monitor usage against plan limits
- **Team Insights**: Understand team activity and productivity
- **Optimization**: Identify top-performing content for replication
