---
title: 'Create Schedule'
openapi: 'POST /api/v1/workspaces/{workspace_id}/schedules'
---

Create a new automated schedule for scenario execution.

<RequestExample>

```bash cURL
curl -X POST "https://api.chanl.ai/api/v1/workspaces/ws_123/schedules" \
  -H "Authorization: Bearer <access_token>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Daily Quality Check",
    "scenarioId": "scenario_123",
    "periodicity": "daily",
    "periodEndCondition": "never",
    "isActive": true
  }'
```

</RequestExample>
