---
title: 'Activate Schedule'
openapi: 'POST /api/v1/workspaces/{workspace_id}/schedules/{id}/activate'
---

Enable a schedule to run automatically.

<RequestExample>

```bash cURL
curl -X POST "https://api.chanl.ai/api/v1/workspaces/ws_123/schedules/schedule_456/activate" \
  -H "Authorization: Bearer <access_token>"
```

</RequestExample>
