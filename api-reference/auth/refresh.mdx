---
title: 'Refresh Access Token'
openapi: 'POST /api/v1/auth/refresh'
---

Refresh an expired access token using a valid refresh token.

Use this endpoint when your access token has expired (after 1 hour) to get a new access token without requiring the user to re-authenticate through Firebase.

<RequestExample>

```bash cURL
curl -X POST "https://api.chanl.ai/api/v1/auth/refresh" \
  -H "Content-Type: application/json" \
  -d '{
    "refresh_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
  }'
```

```javascript JavaScript
const response = await fetch('https://api.chanl.ai/api/v1/auth/refresh', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    refresh_token: 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...'
  })
});

const { data } = await response.json();
localStorage.setItem('access_token', data.access_token);
```

```python Python
import requests

response = requests.post(
    'https://api.chanl.ai/api/v1/auth/refresh',
    json={
        'refresh_token': 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...'
    }
)

data = response.json()['data']
new_access_token = data['access_token']
```

</RequestExample>

<ResponseExample>

{
  "success": true,
