---
title: 'Validate Token'
openapi: 'POST /api/v1/auth/validate'
---

Validate the current access token and retrieve user information.

Use this endpoint to check if a token is still valid and to get current user information and workspace context.

<RequestExample>

```bash cURL
curl -X POST "https://api.chanl.ai/api/v1/auth/validate" \
  -H "Authorization: Bearer <access_token>" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://api.chanl.ai/api/v1/auth/validate', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer ' + accessToken,
    'Content-Type': 'application/json',
  }
});

const { data } = await response.json();
console.log('User info:', data.user);
```

```python Python
import requests

response = requests.post(
    'https://api.chanl.ai/api/v1/auth/validate',
    headers={'Authorization': f'Bearer {access_token}'}
)

user_info = response.json()['data']['user']
```

</RequestExample>
